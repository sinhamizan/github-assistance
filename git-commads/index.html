Git Add/reset Commands

git add --all: We use this command for add all elements.

git add -A: Also we can use this command for add all elements.

git add .: This command also use for add all elements. Different is git add --all or git add -A is add all over the elements(all files in project) but git add . is add all elements in current directory(specific folder where you staying) not all over the elements.

git reset: This command is use for undo the add.

Git Commit Commands

git commit -m "your message": This command for commit

git reset HEAD~: This command do undo add and commit at a time.
